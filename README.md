  - 컬럼을 생성할 수 있습니다.
  - 중복된 컬럼 이름인지 확인합니다.

  - 컬럼 수정 모달에서 기존의 컬럼 이름을 가져옵니다.

  - 컬럼 이름 중복을 확인합니다.
  - 삭제하기버튼을 누르면 삭제 모달이 나타납니다.
  - 컬럼 삭제를 할 수 있습니다.
### 할일 생성
  - 할일을 생성할 수 있습니다.
